-------------------------------------------------
1. GENEL BAKIŞ / OVERVIEW
-------------------------------------------------

TR:
Zip64, Java 25+ için yüksek performanslı, keyfi hassasiyette ondalık sayı kütüphanesidir.
Floating-point (double/float) ile oluşan hassasiyet sorunlarını çözer.
Hem senkron hem de sanal-thread (virtual-thread) ile asenkron işlemleri destekler.
Oyun motoru veya başka bir framework’e bağımlı değildir.

EN:
Zip64 is a high-performance arbitrary-precision decimal library for Java 25+.
It solves precision issues commonly encountered with floating-point (double/float) operations.
Supports both synchronous and asynchronous operations via virtual threads.
It is independent of game engines or any other frameworks.

-------------------------------------------------
2. UYUMLULUK / COMPATIBILITY
-------------------------------------------------

TR:
- Java 25+ tam uyumludur ve tüm özellikleri destekler.
- Java 21 ile çalışabilir, ancak Virtual Threads ve bazı async fonksiyonlarda bug riski vardır.
- Mobil platformlarda Virtual Threads desteklenmediği için async API’leri kullanamazsınız.
- Sadece senkron işlemler güvenlidir.

EN:
- Fully compatible with Java 25+, all features supported.
- Works with Java 21 but with potential bugs in virtual-thread async operations.
- Not supported on mobile platforms due to missing Virtual Threads.
- Only synchronous operations are safe on mobile.

-------------------------------------------------
4. TEMEL KULLANIM / BASIC USAGE
-------------------------------------------------

TR:
```java
Zip64 a = new Zip64("12345678901234567890");
Zip64 b = new Zip64("98765432109876543210");

// Toplama
Zip64 sum = a.add(b);

// Çarpma
Zip64 product = a.multiply(b);

// Sonuçları yazdır
System.out.println("Toplam: " + sum);
System.out.println("Çarpım: " + product);

// Asenkron kullanım (Virtual Threads)
try (Zip64.Async async = Zip64.async()) {
    Zip64 asyncSum = async.add(a, b).join();
    System.out.println("Asenkron Toplam: " + asyncSum);
}
EN:
Java
Zip64 a = new Zip64("12345678901234567890");
Zip64 b = new Zip64("98765432109876543210");

// Addition
Zip64 sum = a.add(b);

// Multiplication
Zip64 product = a.multiply(b);

// Print results
System.out.println("Sum: " + sum);
System.out.println("Product: " + product);

// Asynchronous usage (Virtual Threads)
try (Zip64.Async async = Zip64.async()) {
    Zip64 asyncSum = async.add(a, b).join();
    System.out.println("Async Sum: " + asyncSum);
}
